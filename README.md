# gloc

Run a shell command in all the git repos in a directory.

![gif](https://i.imgur.com/ItUKzAz.gif)


> Idea stolen from [`fabiospampinato/autogit`](https://github.com/fabiospampinato/autogit)


## Install

```
go install
```

## Usage

```
./gloc "git fetch" "./path/to/folder"
```

## Development

### Build

```
go build gloc.go
```
